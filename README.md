# gitstatus

### Install

`brew install Jleagle/gitstatus/gitstatus`

### Help

```
Usage:
  gitstatus [flags]

Flags:
  -a, --all             Show all Repos
  -d, --dir string      Directory
  -f, --filter string   Filter
  -h, --help            help for gitstatus
  -m, --maxdepth int    Max Depth (default 2)
  -p, --pull            Pull Repos
  -s, --short           Short Paths
```
